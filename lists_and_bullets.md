# Lists and bullet points 

To create lists, begin a line with asterisk, or minus:

 * item 1
 * item 2
 * item 3
 
 This is also a list with a minus:
 
 - item 1
 - item 2
 - item 3
 
 Can make sub-points with two spaces before a point:
 * item 1
   * sub-item 1
   * sub-item 2
 * item 2
 
 If you want to create numbered lists:
 
 1. item 1
 2. item 2
 3. item 3
